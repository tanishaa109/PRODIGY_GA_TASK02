# PRODIGY_GA_TASK02 - Text to Image Generator (GenAI)

## 📌 Project Overview
This project is a Generative AI application that converts text prompts into AI-generated images using a pre-trained Stable Diffusion model.

The system takes user input (text prompt) and generates high-quality images using deep learning.

---

## 🚀 Features
- Text-to-image generation using AI
- User input based prompt system
- Pre-trained Stable Diffusion model
- Saves generated images locally
- Works on Google Colab with GPU support

---

## 🧠 Tech Stack
- Python
- PyTorch
- Hugging Face Diffusers
- Transformers
- Google Colab (GPU execution)

---

## ⚙️ How to Run

### 🔹 Option 1: Google Colab (Recommended)
1. Open the Colab notebook
2. Enable GPU runtime
3. Run all cells step by step
4. Enter text prompt to generate images
